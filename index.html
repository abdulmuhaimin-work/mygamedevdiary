A game dev diary

An everyday entry of my progress towards creating my first game and beyond

5/6/2020

5th of June 2020 is the first entry into this barebone game dev diary. I decided to create this to expand my writing and force myself to understand enough to explain game development into the diary. Hey maybe someday I'll be nostalgic when reading this diary.

At this moment in time, I decided too create a custom game in Dota2. Heres the reason why:

  • Im familiar with Dota2 mechanics. As a starting project I think something familiar will be easier to get into
  • Creating a game using Dota2 workshop tools might be limited, but I think thats a good thing for a beginner like me who know nothing about game dev

8/6/2020
Familiarizing with Dota 2 workshop. learned how to modify map, units and abilities using Lua. Really need to refer to dota 2 API page alot. the console is a useful place to see whats happening.
9/6/2020

learn how to create and modify projectiles. "OnXxxx" is functions that listen to specific event occurence. reference can be found on the dev site.

To create custom ability, define the properties in npc_custom_abilities.txt. then create a separate .lua file for the custom abilities behaviour.
